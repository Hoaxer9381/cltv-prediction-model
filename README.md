# Customer Lifetime Value (CLTV) Prediction

## 📌 Overview
This project predicts Customer Lifetime Value using machine learning techniques.

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- LightGBM
- Scikit-learn

## 📊 Approach
- Data preprocessing
- Feature engineering (log + ratio features)
- K-Fold cross-validation
- LightGBM model

## 📈 Results
- Achieved ~0.32 R² (local)
- Optimized for generalization in hackathon

## 🚀 Outcome
Helps identify high-value customers for business decisions.
